# FakeLook
### This project is a Image based social network, created by Nehorai Peisach and Shai Moran for the Sela Learning Center.

## How To Start:
### Client:
```
$ cd ../clinet
```
```
$ npm install
```
```
$ npm start
```

### Server:
```
$ cd ../server
```
```
$ npm install
```
```
$ npm start
```
